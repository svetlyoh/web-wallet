# MALL MAYHEM '99 Art Spec Prompts

Use these prompts when creating or replacing visual assets for the game. Every prompt targets:
- clear silhouette at gameplay scale
- layered detail for close inspection
- nostalgic mall-world storytelling
- visible wear, decals, labels, and light behavior

## Global Style Prompt
`Late-90s arcade mall illustration, top-down readable game asset, bold silhouette, saturated neon accents, CRT glow, reflective tile highlights, slight wear and grime, stickers/price tags/micro-labels, playful but believable materials, high readability at small size, rich details at close look, entertainment tone (not horror).`

## Small-Object Detail Checklist
For each asset, include at least 4 of:
- micro text blocks
- barcode/price tag
- sticker edge lift
- scratch cluster
- smudge/fingerprint
- crack/chip
- seam/fold
- tiny indicator light
- taped note
- reflection hotspot
